Dynomix

...The wave of the future!

Dynomix is a tool to help you get information about your vMix inputs by asking the connected Companion instance very politely.

Fill out the fields below, click "Fetch Data" or press Enter, and you'll get everything you need (and nothing you don't).

Resubmit with a value in the "Filter by input title" field to filter results.

Fill out the form above to populate this area with vMix inputs.

Need help?

Click each question below to open/close the details.

What's my Companion IP?

The IP address of the computer you're using Companion on. You can get this from the Companion program running on the computer. If it's your computer, set Companion to (the default) "All Interfaces" and use localhost or 127.0.0.1. This is the recommended way to use Dynomix!

If Companion isn't running on your computer (but is on another computer on your local network that you can access, you'll need to get a little more advanced. Right now, Dynomix doesn't support that out of the box! You'll need to enable the HTTPS Web Server in your Companion settings to even have a chance, because browsers (wisely) won't connect to mixed-security content.
(This website is secure, and by default the Companion interface and API are not, because they aren't actually on the Internet, but this website tries to talk to them... you see the problem?).

What's my Companion port?

The port that Companion is listening on. Companion defaults to 8000, but check your Companion program to be sure.

What's my vMix name?

The name of the vMix connection inside of Companion. Check the Connections page of the Companion Web Interface!
