What is this? How do I use it?

Dynomix is a tool to help you get information about your vMix inputs by asking the connected Companion instance very politely.

Fill out the fields at the top of the page, click "Fetch Data" or press Enter, and you'll get everything you need: the number, name, and unique identifier for every input.

Resubmit (click "Fetch Data") with an input name in the "Filter by input title" field to filter results.

What's my Companion IP?

The IP address of the computer you're using Companion on. You can get this from the Companion program running on the computer. If it's your computer, set Companion to (the default) "All Interfaces" and use localhost or 127.0.0.1. This is the recommended way to use Dynomix!

If Companion isn't running on your computer (but is on another computer on your local network that you can access, you'll need to get a little more advanced. Right now, Dynomix doesn't support that out of the box! You'll need to enable the HTTPS Web Server in your Companion settings because the web browser (wisely) won't connect to insecure content from a secure origin ( mixed-security content)

By default the Companion interface/API use HTTP because it does not run on the public Internet. This website is on the public Internet.

What's my Companion port?

The port that Companion is listening on. Companion defaults to 8000, but check your Companion program to be sure.

What's my vMix name?

The name of the vMix connection inside of Companion. Check the Connections page of the Companion Web Interface!
