Need help?

Companion IP

The IP address of the computer you're using Companion on. You can get this from the Companion program running on the computer. If it's this computer, you can write localhost.

Companion port

The port that Companion is listening on. Companion defaults to 8000, but you can check the Companion program to be sure.

vMix name

The name of the vMix connection inside of Companion. Check the Connections page of the Companion Web Interface!
